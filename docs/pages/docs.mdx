# Getting Started with the Public AI Gateway

Welcome to the Public AI Gateway! This guide will help you get started with accessing open-source AI models through our unified API.

## What is the Public AI Gateway?

Public AI Gateway is a unified access point for multiple public AI models and inference providers. We handle load balancing, traffic routing, and failover across our network of partners to ensure reliable access to open-source AI infrastructure.

## Quick Start

### 1. Create an Account

To get started, you'll need to create an account and obtain an API key.

<Stepper>
1. Click on the _Login_ button on the top right of this portal
2. Sign up using single sign on 
3. After sign up, click on your name to reveal the dropdown menu
4. Click on _API Keys_
5. Click _Create API Key_
6. Enter a name for your API Key and select an expiration time
7. Click _Generate Key_
</Stepper>

### 2. Making Your First Request

Once you have an API key, you can start making requests to our AI models. Here's a simple example:

```bash
curl -X POST https://api.publicai.co/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "model": "swiss-ai/apertus-70b-instruct",
    "messages": [
      {
        "role": "user",
        "content": "Hello! Can you help me understand open-source AI?"
      }
    ]
  }'
```

## Authentication

All API requests require you to authenticate using an API key. Include your key in the Authorization header:

```
Authorization: Bearer YOUR_API_KEY
```

## Rate Limits

To ensure fair usage across all users:
- **During Swiss AI Weeks:** 20 requests per minute
